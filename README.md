# Desafio Listagem YouTube Grid

Olá, esse projeto foi realizado como parte da trilha de Formação Desenvolvedor CSS da plataforma DIO no qual o desafio consistia em desenvolver a listagem da página inicial do YouTube utilizando conceitos de Grid do CSS. A ideia era organizar o layout usando Grid com o intuito de facilitar a criação de colunas e linhas que contribuem para o posicionamento dos elementos de forma adaptável e inteligente.

[Clique aqui](https://arturbarbuzano.github.io/desafio-listagem-youtube-css/) para acessar o resultado da minha listagem da página inicial do YouTube.
